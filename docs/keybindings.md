# 操作介面與快速鍵規劃

> 對齊先前中文化範本(u3-cht / u4-cht)與全域離開鐵則。
> 原則:**ESC 只 cancel/back,F10(或 Ctrl+Q)才離開,離開前跳 是/否 並自動存檔**。

## 1. 移動與視角

| 鍵 | 動作 | 上游現況 |
|---|---|---|
| ↑ ↓ ← → | 世界地圖/城鎮移動;地牢中前進/後轉/左右轉 | 已處理(UP/DOWN/LEFT/RIGHT) |
| (數字鍵盤) | 八方向移動(後續) | 未 |

## 2. 指令(沿用 U1 經典單鍵,顯示中文)

| 鍵 | U1 指令 | 中文顯示 | 現況 |
|---|---|---|---|
| A | Attack | 攻擊 | 已處理(`SDLK_a`) |
| E | Enter | 進入 | 已處理(`SDLK_e`) |
| K | Klimb(上下樓梯) | 攀爬 | 已處理(`SDLK_k`) |
| C | Cast | 施法 | 未 |
| G | Get | 拾取 | 未 |
| T | Talk / Transact | 交談 | 未 |
| R | Ready weapon | 備武 | 未 |
| Z | Ztats(角色狀態) | 屬性 | 未 |

> 單鍵指令維持英文鍵位(玩家肌肉記憶),畫面回饋訊息全中文。對照表是翻譯介面,不改鍵位。

## 3. 系統鍵(全域鐵則,所有畫面一致)

| 鍵 | 動作 |
|---|---|
| `ESC` | 取消 / 返回上一層(**永不直接離開遊戲**) |
| `F10` 或 `Ctrl+Q` | 離開遊戲 → 跳「確定離開?是/否」→ 是則**自動存檔**後退出 |
| `F5` | 手動存檔(可選) |
| `+` / `-` | 視窗整數倍縮放(可選) |
